# Preliminary Thesis: Zero-Day Malware Detection in Docker Containers

This project investigates a self-supervised learning approach for detecting zero-day
malware in Docker containers using static filesystem analysis.

## Structure
- notebooks/: Experiments and analysis
- data/: Raw and processed container data
- scripts/: Data preprocessing scripts
- results/: Figures and logs

## Status
Environment setup complete. Data extraction and preprocessing in progress.
